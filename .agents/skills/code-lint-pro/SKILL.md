# Code Lint Pro

## Overview
A fast, lightweight linting tool for identifying common bad practices in TypeScript and general code.

## When to Apply
- Before committing code
- During code review
- When debugging "any" type issues or console leaks

## How to Use
- Analyze the code for "console.log" statements
- Identify any "any" types in TypeScript code

## Rules
- **No Console Logs**: Always remove logs before production
- **Strong Typing**: Avoid 'any' type to ensure type safety
